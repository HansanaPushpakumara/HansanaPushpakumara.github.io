Ceylon Cart - Online Farm Products Selling System

This project is an e-commerce web application which provides a good platform to improve the interconnection among the farms and customers. Customers like food processing companies and supermarkets can place orders within the system. Then suppliers can get to know about orders through SMS and reserve them via application easily. Suppliers can make card payments for the suppliers. This project was implemented after doing a case study on related target groups.

Technologies used
Laravel, HTML, CSS, JavaScript, jQuery
